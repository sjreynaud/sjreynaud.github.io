
# FSGS Kidney Disease Study

This project analyzes clinical and survival data from patients diagnosed with FSGS Kidney Disease.

## Structure
- `notebooks/`: Data exploration, preprocessing, modeling, visualization
- `src/`: Modular pipeline scripts
- `reports/`: Figures, tables, publication drafts
- `docs/`: Project overview, methods, references
- `tests/`: Unit tests for pipeline components

## Usage
1. Mount Google Drive in Colab
2. Load and preprocess `fsgs_dataset_cleaned.csv`
3. Run survival modeling and generate reports

## Requirements
See `environment.yml` for dependencies.
